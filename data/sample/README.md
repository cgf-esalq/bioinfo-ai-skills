# 📁 Dados de Exemplo

Este diretório contém dados pequenos para uso nos exercícios dos workshops.

## Dados disponíveis

| Arquivo | Descrição | Tamanho |
|---------|-----------|---------|
| `bovine_variants_sample.vcf` | VCF de exemplo com ~100 variantes de *Bos taurus* (chr14) | ~50 KB |
| `expression_matrix_sample.tsv` | Matriz de expressão simplificada (10 genes × 20 amostras) | ~5 KB |
| `genotypes_sample.tsv` | Genótipos para análise de eQTL (10 SNPs × 20 amostras) | ~3 KB |

## Origem dos dados

- **VCF**: Subconjunto de variantes públicas do 1000 Bull Genomes Project, região do gene DGAT1 (BTA14)
- **Expressão**: Dados simulados baseados em padrões reais de expressão em fígado bovino
- **Genótipos**: Dados simulados para fins didáticos

> ⚠️ Estes dados são **apenas para fins educacionais** e não devem ser usados para conclusões científicas.

## Como baixar (se necessário)

Os dados de exemplo já estão incluídos no repositório. Para dados maiores, consulte o [guia de links úteis](../../resources/useful_links.md).
